# add-delivery-intelligence-pm

Phase 5.1: the full PM/PO delivery-intelligence metric set — cycle/lead-time percentiles,
aging WIP, throughput & net-flow trends, backlog forecasting, work-mix, quality signals
(reopen rate, first-response), milestone burn-up + projected completion, and team-load /
bus-factor — built on a forward-only metrics time-series and lightly enriched issue/milestone
capture. Exposed over MCP tools, REST endpoints, and a Delivery dashboard view.
